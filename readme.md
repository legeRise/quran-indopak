# IndoPak Quran Resources

This repository contains a handpicked set of resources to render the Quran in **IndoPak Nastaleeq style**:

- `indopak-ayah-by-ayah.zip` – Quran text by ayah  
- `indopak-word-by-word.zip` – Quran text by word  
- `indopak-taj-company-16-line-layout.zip` – 16-line Mushaf layout  
- `indopak-nastaleeq-font.zip` – IndoPak Nastaleeq font for proper rendering  

All these resources are available on [QUL Tarteel](https://qul.tarteel.ai). They have been collected here for convenience for anyone looking to build a **specific IndoPak-style Mushaf** for web, apps, or other digital use.
